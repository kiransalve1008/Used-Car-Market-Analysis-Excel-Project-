**Used Car Market Analysis (Excel Project)**

**Project Overview**
This Excel-based data analysis project explores the used car market using a detailed dataset of pre-owned vehicles. The analysis focuses on understanding pricing trends, brand demand, mileage impact, vehicle attributes, and buyer preferences to derive actionable market insights.
________________________________________
Objectives
•	Analyze factors affecting used car prices
•	Identify high-demand brands and models
•	Understand the impact of mileage, year, body type, and transmission
•	Support data-driven decisions for buyers and sellers
________________________________________
Dataset Summary
•	Rows: 7,847
•	Columns: 20
•	Key fields include: Make, Model, Year, Price, Mileage, Body Type, Transmission, Fuel Type, Drivetrain, Location
(Dataset cleaned and anonymized for learning purposes)
________________________________________
Tools & Techniques Used
•	Microsoft Excel
•	Pivot Tables & Pivot Charts
•	Conditional Formatting
•	Filters & Sorting
•	Find & Replace
•	Data Cleaning & Formatting
•	Exploratory Data Analysis (EDA)
________________________________________
Steps Followed
1.	Data understanding (rows, columns, data types)
2.	Data cleaning & formatting
3.	Removal of blanks and validation of duplicates
4.	Pivot tables for summary analysis
5.	Visual analysis using charts
6.	Insight generation & interpretation
________________________________________
Key Insights
•	Luxury brands like Ferrari, BMW, and Mercedes-Benz command premium prices
•	Low-mileage vehicles are priced significantly higher
•	Recent manufacturing years (2019–2020) show higher resale value
•	Coupe body type is more popular than convertible in premium cars
•	Rear-Wheel Drive (RWD) and automatic transmission are preferred
•	High-performance models attract enthusiast buyers
________________________________________
Visual Analysis Included
•	Brand-wise price comparison
•	Year-wise price trend
•	Mileage vs Price relationship
•	Body type distribution (Pie Chart)
•	Conditional formatting for price ranking
________________________________________
Business Takeaways
•	Stock high-demand luxury brands
•	Promote low-mileage and performance vehicles
•	Use competitive, data-backed pricing
•	Highlight vehicle condition and history for buyer trust

NOTE-This project is created for educational purposes using anonymized data.
